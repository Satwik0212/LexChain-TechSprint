# LexChain Frontend

A modern, secure, and responsive frontend for the LexChain legal tech platform, built with **React + Vite** and **Tailwind CSS**.

## Features

*   **Secure Login**: Sleek split-screen authentication page.
*   **Dashboard**: Overview of contract analysis tools and history.
*   **Contract Analysis**: Upload interface associated with AI risk assessment.
*   **Analysis Results**: Detailed clause-by-clause risk breakdown.
*   **Blockchain Verification**: Proof of immutability and integrity checks.
*   **History**: Searchable archive of past analyses.

## Tech Stack

*   **React** (v18)
*   **Vite** (Build tool)
*   **Tailwind CSS** (v3.4 - Styling)
*   **React Router** (Navigation)

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd lexchain-frontend
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in Browser**:
    Visit `http://localhost:5173/` (redirects to `/login`).

## Project Structure

*   `src/pages/`: Individual page components (Login, Dashboard, Analysis, etc.)
*   `src/components/`: Reusable UI components.
*   `src/index.css`: Global styles and Tailwind configuration.
